Frequently asked questions

What border styles are available?

You can choose from a solid color border, rounded corners, a classic polaroid-style frame with extra space at the bottom, a dashed outline, or a double-line frame.

Can I change the border color and thickness?

Yes, use the color picker to set any frame color and the width slider to adjust the border thickness from a thin 2px line to a bold 120px frame.

Does the border increase the image dimensions?

Yes, the border is added around the original photo, so the final downloaded image is larger than the source by the border width on each side.

Will the polaroid frame change the image's aspect ratio?

Yes, the polaroid style adds extra space below the photo for that classic instant-camera look, which changes the overall canvas height while keeping your original photo unmodified.
