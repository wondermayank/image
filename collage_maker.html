Frequently asked questions

How many images can I add to the collage?

You can add up to 9 images. The layout adapts based on how many images you add, with options for 1×2, 2×2, 2×3, and 3×3 grids.

Will the images be cropped?

Yes — each cell uses object-fit cover logic, so images are centered and cropped to fill their cell evenly. This keeps the grid looking clean and uniform.

Can I change the gap between images?

Yes. Use the gap slider to set the spacing between grid cells from 0 (no gap) to 20 pixels.

What format is the download?

The collage downloads as a PNG file. The output resolution depends on the cell size and number of images in the grid.
