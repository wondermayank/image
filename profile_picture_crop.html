Frequently asked questions

What size should a profile picture be?

Common profile picture sizes range from 200×200 to 512×512 pixels, depending on the platform. Most exam portals and professional platforms ask for a square photo, often around 200×200 pixels.

Should I use a circle or square crop for my profile picture?

Square crops work for most exam form uploads since portals typically expect a rectangular file even if it's displayed in a circle later. Circle crops are useful when you need to preview exactly how a platform like LinkedIn or a chat app will display your photo.

Does a circle crop save as a circular image file?

Yes, circle crop mode exports a PNG with transparent corners outside the circle. If your destination requires a plain square JPEG, use square crop mode instead.

Can I set a custom output size in pixels?

Yes — type any value into the output size field, or pick one of the common presets (128, 200, 256, 400, 512px). The output is always a perfect square or circle at that exact pixel size.
