Frequently asked questions

What font does the meme text use?

The tool uses Impact font with a white fill and black stroke — the classic internet meme style. Font size is configurable with the slider.

Can I customize the text color?

Yes. You can change both the text fill color and the outline color using the color pickers next to the text fields.

Is there a character limit?

There's no hard character limit, but very long text is auto-scaled down to fit within the image width. Shorter, punchier captions work best.

What format is the download?

Your meme downloads as a PNG at the original image resolution with the caption baked in.
