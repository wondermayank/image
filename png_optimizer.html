Frequently Asked Questions

How does PNG optimization work?

PNGs can contain redundant metadata, unnecessary color depth, and inefficient encoding. This tool re-encodes the image with optimal settings and strips non-essential chunks.

What does reducing colors do?

PNG supports palette-based encoding with fewer colors. Reducing from millions to 256 or 128 colors can dramatically shrink file size for graphics, icons, and illustrations — but may affect photos.

Is this truly lossless?

With "Original" color mode and no color reduction, the re-encoding is visually lossless. Color reduction is lossy — use it for graphics, not photographs.

What metadata gets stripped?

EXIF data, text chunks (author, description, comments), timestamps, and color profile metadata. Pixel data is preserved.
